# Encriptador Challenge-Oracle-One
pagina web que permite encriptar o desencriptar un texto ingresado por el usuario siguiendo reglas de encriptación establecidos.

Proyecto que forma parte del Spring 01: Construye un encriptador de texto con JavaScript.

## Desarrollado con:
HTML - CSS- JAVASCRIPT

## Codigo

+La letra "e" es convertida para "enter"
+La letra "i" es convertida para "imes"
+La letra "a" es convertida para "ai"
+La letra "o" es convertida para "ober"
+La letra "u" es convertida para "ufat"

## Requisitos

+La página debe tener campos para la inserción del texto que será encriptado o desencriptado, y el usuario debe poder escoger entre esas dos opciones.

+Debe ser posible convertir una palabra a su versión encriptada y también devolver una palabra encriptada a su versión original.

+Debe funcionar solo con letras minúsculas.

+No deben ser utilizadas letras con acentos ni caracteres especiales.El resultado debe ser mostrado en la pantalla.
